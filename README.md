# 💰 Finance Tracker

A simple and efficient **Finance Tracker** application to manage your daily income and expenses. This project helps users keep track of their financial activities and analyze spending habits.

---

## 🚀 Features

* ➕ Add income and expense transactions
* 📊 View total balance, income, and expenses
* 🗑️ Delete transactions
* 📅 Track daily financial records
* 📈 Simple and clean UI
